Delay OTA Guide written by Flower

You can update using DelayOTA to versions for up to 90 days after a new version is released.

Opening notes

You will need to wipe your device when you supervise your device if you're unjailbroken, so I suggest you take a backup while unsupervised. Either using iCloud or using iMazing/Apple configurator 2) DON'T BACKUP IF YOU'RE JAILBROKEN!

Your device has a chance to enter a recovery loop if you use this method, if you somehow do you'll have to update to what's signed/or use blobs.

For Jailbroken devices: (Easiest)

1. Install supervised enabler in your package manager by adding this repo, and then searching "supervised enabler" in the search area.

  • For iOS 12 and lower
    • Since you can't install Supervised Enabler you'll have to edit the file manually. Go on your mobile browser and press here!! It will show you where the file in Filza File Manager is. Now set the IsSupervised argument in the plist to YES.

2. Once done install the profile you want to go to here (It will show the current available versions right away).

3. Go to settings and search for the profile, either press on Profile Downloaded on the main page or go to Settings -> General -> Profiles & Device Management.

4. Press on install when you press on the profile.

5. Next, Restore RootFS before upgrading, to ensure nothing messes up when you update. (This wont erase any personal data i.e photos & messages).

  • Unc0ver
    • Reboot the device then open unc0ver. Tap the gear icon and enable the Restore RootFS toggle, then tap Restore RootFS at the main screen.

  • Chimera / Odyssey / Taurine
    • Reboot, open Chimera / Odyssey / Taurine, toggle on Restore RootFS, then tap Jailbreak.

  • Checkra1n / Odysseyra1n
    • Open the checkra1n app on your home screen and tap Restore System. If the icon has disappeared or you get Restore Error: General, reboot and re-jailbreak and try again.

6. Once done it will reboot your phone and your jailbreak will be removed, go to Settings -> General -> Software Update and see if the version you chose is there. (If it doesnt show up reboot your phone or do all the steps all over again)

7. Press Download and Install and you should see the progress bar move.

You're Done!

If you're done it correctly your phone should update like any other OTA update.

For Non-Jailbroken devices: (Hardest)

  • Windows: Using iMazing: (Paid)

      • 1. Download and install iMazing from here

      2. Go through the setup and press the Supervision button on the right.

      3. It will show you what you can do with Supervision so press Next.

      4. It will now show the Supervision screen showing Supervise at the top. Press the button right below Organization. Press Choose.

      5. A new window will appear showing the Organizations, Press the + Button on the bottom left corner of that window.

      6. Add a name (It can we whatever you want), and press Save, now press Choose.

      7. Press Next and it will warn you that it will erase all contents, type the number it shows you in the text box to confirm, then press Next (It will now erase your phone and supervise it, it wont update the phone.)

      8. Your phone will reboot will probably show a setup screen, go set it up.

      9. Once done install the profile you want to go to here (It will show the current available versions right away).

      10. Go to settings and search for the profile, either press on Profile Downloaded on the main page or go to Settings -> General -> Profiles & Device Management.

      11. Reboot your phone and then go to Settings -> General -> Software Update, you should see the version you picked there.

      12. Press Download and Install and you should see the progress bar move.

  • MacOS: Using Apple Configurator 2:

      • 1. Go to the App Store on your Mac and search for Apple Conifgurator 2, install it from there.

      2. Do the setup like usual and press Get started if you haven't already.

      3. Press on your device, and press Prepare and select Manual supervision, press Next.

      4. Choose Do not enroll in MDM, press Next.

      5. Go to Organization and press New Organization. This part is optional, either sign into your Apple Id or press Skip.

      • If you chose skip, add a name, and press Next.

      6. Press Generate a supervision identity (in case you've done this before, pick Choose existing supervision identity option).

      7. Configure iOS Setup Assistant as you choose, it doesnt really matter what you do here.

      8. Press Prepare (It will now erase your phone and supervise it, it wont update the phone).

      9. Your phone will reboot will probably show a setup screen, go set it up (If you chose not to in the Configure iOS Setup Assistant you can skip this step).

      10. Once done install the profile you want to go to here (It will show the current available versions right away).

      11. Go to settings and search for the profile, either press on Profile Downloaded on the main page or go to Settings -> General -> Profiles & Device Management.

      12. Reboot your phone and then go to Settings -> General -> Software Update, you should see the version you picked there.

      13. Press Download and Install and you should see the progress bar move.

You're Done!

If you done it correctly your phone should update like any other OTA update.

Closing notes:

If you wish to restore from a backup from iMazing/Apple Configurator 2 after updating use DelayOTA follow below (Note that restoring from a backup will remove your supervision).

  • Windows: Using iMazing: (Paid)

      • 1. In iMazing, press Restore a Backup on the Right.

      2. iMazing will probably automatically select your backup, press Choose.

      3. The Restore Options page will come up, choose what you want to restore from and press Next.

      4. You'll need to turn off Find My iPhone for this step, Go to Settings -> Apple ID -> your device -> Find My and turn it off.

      5. Once done you'll get a Confirm Restore screen, press Restore.

  • MacOS: Using Apple Conifgurator 2:

      • 1. In Apple Conifgurator 2, select your device you want to erase.

      2. Do one of the following:

      • Choose Actions -> Restore from Backup.

      • Control + Left Click (or Right Click) your selected device and choose Restore from Backup.

      3. Choose your backup and press Restore

      • If you get a prompt saying you need to turn off Find My in Settings>Apple ID -> your device -> Find My , turn it off.

You're Done!

If you wish to Erase and Un-Supervise after updating your device follow the steps below (Doing this will UnSupervise the device and Erase it).

  • Windows: Using iMazing: (Paid)

      • 1. On the right scroll all the way down and press Erase All Content (Doing this will UnSupervise the device and Erase it).

      3. Once booted go through the normal setup and your phone should be unsupervised.

  • MacOS: Using Apple Conifgurator 2:

      • 1. Select your device you want to Erase.

      2. Do one of the following:

      • Choose Actions -> Advanced -> Erase All Content and Settings.

      • Control + Left Click (or Right Click) your selected device and choose Advanced -> Erase All Content and Settings.

      3. Once booted go through the normal setup and your phone should be Un-Supervised.

You're Done!

If you done it correctly should either have restored from a backup or has been erased entirely.
